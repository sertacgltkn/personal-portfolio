### Summary

This project has been prepared as a graduation project of React Bootcamp organized in cooperation with [Kodluyoruz](https://kodluyoruz.org/tr/kodluyoruz/) and [FÄ±rat Development Agency](https://fka.gov.tr).

While preparing this project, I tried to use React Hooks, Context structure, libraries such as React Router Dom, Bootstrap, Material UI. Besides, e-mail connection was made with Email.js.

### Getting started

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. Visit [http://localhost:3000](http://localhost:3000) to see the app running
